Chess Simulator

Java

This Chess project allows user to simulate a game of chess with yourself or someone else.

Easy-to-read chess boards and status messages allow for a user-friendly experience.

I implemented object-oriented concepts like enum, loops, and functions to program the terminal-based game.

Through Java, I incorporated all chess features such as checking for illegal moves, check, and checkmate scenarios
