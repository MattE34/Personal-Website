This Amazon project included data selection and preprocessing, rating predictions, and item recommendations.

Matrix factorization (SVD), collaborative filtering, and other machine learning topics were implemented to provide accurate results.

MAE, RMSE, precision, recall, F-measure, and NDCG metrics were measured to evaluate and optimize the model's accuracy.

I utilized the NumPy, Pandas, and Sklearn Python libraries in Jupyter Notebook for this project.
